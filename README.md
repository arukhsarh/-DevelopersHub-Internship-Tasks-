Objective: To group customers based on their spending habits to help businesses design better marketing strategies.
Algorithm Used: K-Means Clustering.
Key Insights:

Optimal Clusters: Using the Elbow Method, we found that 3 to 5 clusters are ideal.

Low Spenders: Customers with low total bills and tips.

Average Spenders: Middle-range customers.

High Spenders: Premium customers who spend more and tip well (Target Audience).

# Task 5: Interactive Business Analytics Dashboard 📊
Objective
To develop a user-friendly, interactive dashboard for a retail business (Global Superstore style) that allows managers to track key performance indicators (KPIs) like Sales and Profit across different regions and categories.

# Tools & Libraries Used
Streamlit: For building the interactive web interface.

Pandas: For data manipulation and filtering.

Plotly Express: For creating dynamic and responsive charts.

Localtunnel/Pyngrok: To host the dashboard directly from Google Colab.

# Key Features
Dynamic KPIs: Real-time calculation of Total Sales, Total Profit, and Profit Margin.

Interactive Filters: A sidebar with a multi-select filter for Regions, allowing users to drill down into specific data segments.

Data Visualization: A bar chart that visually compares sales performance across different product categories (Technology, Furniture, etc.).

Data Preview: A detailed table view at the bottom showing the filtered raw data for transparency.

# How to Run
Open the provided Jupyter Notebook in Google Colab.

Install the required libraries (streamlit, plotly).

Run the code cells to generate the app.py file and launch the tunnel.

Access the dashboard via the generated URL.
