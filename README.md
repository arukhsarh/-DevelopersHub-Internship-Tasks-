# Task 2: Customer Segmentation using Unsupervised Learning
Objective: To group customers based on their spending habits to help businesses design better marketing strategies.
Algorithm Used: K-Means Clustering.
Key Insights:

Optimal Clusters: Using the Elbow Method, we found that 3 to 5 clusters are ideal.

Segments identified:

Low Spenders: Customers with low total bills and tips.

Average Spenders: Middle-range customers.

High Spenders: Premium customers who spend more and tip well (Target Audience).
